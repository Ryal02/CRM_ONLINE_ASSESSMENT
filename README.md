## Project Setup Guide

#Prerequisites
Ensure you have the following installed on your system:
  Docker
  PHP 8.2
  Node.js v20
  Composer
  Yarn

##Clone the Repository
First, Create a folder to organize the project
Second, clone the project repository inside the folder you created:
